ORU R01 Lab Result Profile

ORU R01 messages should include MSH, PID, PV1, ORC, OBR, and one or more OBX segments. OBX-2 carries the value type, OBX-3 carries the observation identifier, OBX-5 carries the result value, OBX-6 carries units, OBX-7 carries reference range, OBX-8 carries abnormal flag, and OBX-11 carries the result status.

ADT Visit Profile

ADT messages should include MSH, EVN, PID, and PV1. ADT A01 represents admission or registration. ADT A03 represents discharge. PV1-2 carries patient class and PV1-19 carries the visit number.

Radiology Report Profile

Radiology ORU messages should use OBR for the study and OBX text values for the report narrative. The report should preserve provider, order, patient, study description, and final status context.
